Melhorar os filtros de `Data inicial` e `Data final` com calendário visual e navegação rápida por dia.

## Escopo
- `frontend/src/views/ManifestsView.vue`

## Resultado
Os campos de data passaram a ter:
- seleção via calendário;
- botão para dia anterior;
- botão para dia posterior;
- botão dedicado para abrir calendário.

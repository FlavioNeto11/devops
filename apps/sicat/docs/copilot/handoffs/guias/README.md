# Guias de Handoff

Esta pasta concentra documentação transversal de orquestração e execução de handoffs.

## Conteúdo
- Guias rápidos, índices e fluxogramas
- Comparativos entre abordagens de handoff
- Conclusões e resumos de implementação de orquestração
- `legado/` para relatórios e resumos históricos que não são mais porta de entrada

## Observação
Artefatos de execução de um DL específico ficam em `docs/copilot/handoffs/DL-XXX/`.

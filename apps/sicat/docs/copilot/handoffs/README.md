# Handoffs Copilot

Esta pasta organiza a documentacao de handoffs da camada Copilot por tipo de artefato.

## Estrutura

- guias/: guias operacionais canônicos e referencias transversais.
- guias/legado/: relatorios e resumos historicos preservados, mas nao prioritarios para discoverability atual.
- DL-XXX/: artefatos ligados a uma decisao ou entrega especifica.
- DL-XXX/execution/: materiais operacionais e evidencias de execucao daquele DL quando fizer sentido.

## Distincao

Se voce estiver seguindo uma demanda corrente, leia primeiro docs/handoffs/<work_id>/. Se estiver estudando a evolucao da camada Copilot, use este diretorio.

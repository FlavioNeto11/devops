🚀 FEAT: Prompt unificado /handoff para orquestração multi-camada

## Resumo da Mudança

Implementado **prompt único `/handoff`** que orquestra os 3 prompts especializados 
(execute, plan, track) de forma automática e transparente.

### Antes (Complexo)
- 3 prompts separados: /handoff-execute, /handoff-plan, /handoff-track
- Usuário precisava chamar em sequência
- Possibilidade de chamar fora de ordem

### Depois (Simples) ⭐
- 1 prompt unificado: /handoff [descrição]
- Orquestração 100% automática
- Impossível chamar errado

## Como Usar

```bash

@workspace #executor-handoffs

# 2. Usar prompt unificado
/handoff Implemente JWT com refresh tokens

# 3. Acompanhar
docs/copilot/13-decision-log.md
→ Feature pronta em 2-6 horas
```

## Exemplos

### Simples (45 min)
```
/handoff Adicione campo "internalNotes" opcional em manifestos
```

### Média (2-3 horas) - MAIS COMUM
```
/handoff Implemente autenticação JWT com refresh tokens
```

### Complexa (4-6 horas)
```
/handoff Redesenhe fluxo de cadastro para suportar CNAE múltiplos
```

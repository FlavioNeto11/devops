# Legado

Esta pasta agrupa documentos preservados por rastreabilidade historica que nao fazem mais parte da trilha principal de discoverability.

## Uso

- Consulte estes arquivos apenas quando precisar recuperar contexto operacional antigo, validar uma entrega historica ou entender por que um fluxo antigo foi documentado daquela forma.
- Nao use esta pasta como fonte canônica para comportamento atual sem confirmar primeiro em docs/, docs/copilot/, docs/handoffs/ e no codigo.

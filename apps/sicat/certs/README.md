# Certificados CETESB

Esta pasta contém artefatos de certificado usados para conectividade com endpoints CETESB.

## Política

- O uso de certificados deve ser coerente com os fluxos reais observados em `docs/cetesb/`.
- Não armazenar segredos privados nesta pasta.
- Ajustes de TLS devem ser documentados com referência às evidências em `docs/cetesb/`.
